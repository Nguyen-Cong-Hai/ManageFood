# Chức năng chính dự án

- Quản lý authentication bằng JWT Access Token và Refresh Token.
- Có 3 role: Admin, Nhân viên, Khách hàng.

Admin

- Quản lý tài khoản
- Quản lý nhân viên
- Quản lý món ăn
- Quản lý bàn ăn
- Quản lý hóa đơn gọi món
- Thống kê doanh thu

Nhân viên

- Quản lý tài khoản cá nhân
- Quản lý hóa đơn gọi món
- Thống kê doanh thu

Khách hàng

- Xem menu
- Đặt món ăn bằng QR Code
